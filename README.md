# Studio Event Timer

Sistema de cronômetro para eventos com gerenciamento de tempos e personalização visual.

## Características

- Cronômetro com contagem regressiva e progressiva
- Gerenciamento de múltiplos eventos
- Upload de logo personalizado
- Interface responsiva e moderna
- Armazenamento local dos dados

## Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Desenvolvimento

### Requisitos

- Node.js (recomendado via [nvm](https://github.com/nvm-sh/nvm#installing-and-updating))
- npm ou bun

### Instalação

```sh
# Clone o repositório
git clone <YOUR_GIT_URL>

# Entre no diretório
cd studio-event-timer

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

## Deploy

O projeto pode ser facilmente deployado em qualquer plataforma que suporte aplicações React estáticas, como:

- Vercel
- Netlify
- GitHub Pages
- Docker (Dockerfile incluído)
